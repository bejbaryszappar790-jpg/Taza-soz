# Taza-Soz Backend

Проект представляет собой ИИ-юриста с RAG-архитектурой для анализа договоров.

## Установка
1. Создать окружение: `python3 -m venv venv`
2. Активировать: `source venv/bin/activate`
3. Установить зависимости: `pip install -r requirements.txt`

## Запуск
`python3 -m uvicorn backend.main:app --reload`

## API Документация
Swagger UI доступен по адресу: `http://127.0.0.1:8000/docs`
